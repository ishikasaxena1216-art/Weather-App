# 🌦️ Weather App

This is a simple Weather App that shows real-time weather information for any city.

## 🚀 Features
- Search weather by city name
- Shows temperature, humidity, and weather condition
- Clean and simple UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Weather API

- Building a responsive UI
